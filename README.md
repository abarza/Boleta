Boleta
======

Esta simple aplicación te ayuda a calcular el monto que debes ingresar al momento de hacer una Boleta de Honorarios en el Servicio de Impuestos Internos (SII) de Chile. Te muestra valores líquidos y brutos, según hayas negociado con tu cliente. Realicé la aplicación con fines educativos, para practicar y aprender más sobre Android.
